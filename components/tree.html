基本使用 #

需要传入一个 data 的树形结构来展示

Node 1
Node 1-1
Node 1-1-1
Node 1-1-2
Node 1-1-3
Node 1-2
Node 1-3
Node 2
显示代码
html
<template>
  <f-tree :data="data" />
</template>

<script lang="ts" setup>
  const data = [
    {
      label: 'Node 1',
      children: [
        {
          label: 'Node 1-1',
          children: [
            { label: 'Node 1-1-1' },
            { label: 'Node 1-1-2' },
            { label: 'Node 1-1-3' }
          ]
        },
        { label: 'Node 1-2' },
        { label: 'Node 1-3' }
      ]
    },
    { label: 'Node 2' }
  ]
</script>
